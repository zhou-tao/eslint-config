# @toryz/eslint-config-react

[![npm](https://img.shields.io/npm/v/@toryz/eslint-config-react?color=1dd1a1&label=)](https://npmjs.com/package/@toryz/eslint-config-react)

lint for vite + react

### Install

```bash
pnpm add eslint @toryz/eslint-config-react -D
```

### Usage

##### add `.eslintrc.json`
```jsonc
{
  "root": true,
  "extends": "@toryz/react",
  "rules": {
    // your custom rules
  }
}
```
