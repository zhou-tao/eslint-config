# @toryz/eslint-config-basic

[![npm](https://img.shields.io/npm/v/@toryz/eslint-config-basic?color=1dd1a1&label=)](https://npmjs.com/package/@toryz/eslint-config-basic)

lint for vanilla

### Install

```bash
pnpm add eslint @toryz/eslint-config-basic -D
```

### Usage

##### add `.eslintrc.json`
```jsonc
{
  "root": true,
  "extends": "@toryz/basic",
  "rules": {
    // your custom rules
  }
}
```
